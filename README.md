# ✈️ Airline Flight Price Exploratory Data Analysis

## Project Overview

This project performs an extensive Exploratory Data Analysis (EDA) on approximately **300,000 airline flight records** to understand the major factors affecting airline ticket prices.

The analysis investigates how variables such as airline, flight duration, number of stops, departure time, and source city influence airfare through statistical summaries and visualizations.

---

## Objectives

The project answers the following business questions:

- Which airlines operate the most flights?
- Which airline has the highest average ticket price?
- Which source city has the highest flight frequency?
- Does the number of stops affect ticket price?
- Does flight duration influence airfare?
- Which departure period is the most expensive?
- What is the distribution of ticket prices?
- What factors contribute to expensive tickets?

## Key Insights

- Premium airlines charge considerably higher fares.
- Ticket prices generally increase with flight duration.
- Flights with additional stops are usually more expensive.
- Ticket prices are positively skewed due to premium fares.
- Airline selection has a stronger effect on ticket prices than flight duration.
- Major metropolitan cities contribute the highest number of flights.
